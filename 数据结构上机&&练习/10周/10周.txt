1.	二叉树的顺序存储
【问题描述】

用静态二叉链表存储一个二叉树，

输入形式：

从键盘输入若干个节点及左右孩子的下标，以#作为输入结束

要求实现以下功能：

1.给定关键字，输出该字符的双亲和左右孩子结点；如果无双亲，输出NO PARENT;若不存在这样的字符，输出ERROR

2.给定关键字， 从二叉树中删除以该节点为根节点的子树，并输出删除之后的静态二叉链表。



【样例输入】

A 2 3

B 0 4

C 5 6

D 0 0

E 7 8

F 0 0

G 0 0

H 0 0

#

D  // 查询该结点的父亲节点和左右孩子节点

C //删除以C 为根节点的子树


【样例输出】

B 

no lchild

no rchild



A 2 0

B 0 3

D 0 0


2.	求二叉树某结点的最左孩子结点及其所在层次
【问题描述】

假设二叉树中的每个结点值为单个字符，采用二叉链表存储结构存储,按照先序遍历序列创建二叉树。

从键盘输入结点，输出该结点的最左孩子及其所在层次。

若该结点没有左孩子，则输出自身。

若不存在结点，输出NOT EXISTED

【输入形式】

AB##CDF###E#G##

C
【输出形式】

F

4
